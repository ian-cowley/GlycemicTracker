# Security Policy

We take the security of GlycemicTracker seriously. If you believe you have found a security vulnerability, please follow the guidelines below to report it responsibly.

## Supported Versions

The security policy applies only to the latest version of the codebase on the main branch:

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

**Please do not report security vulnerabilities via public GitHub issues.** 

If you discover a vulnerability, please report it privately:
1. Email details of the issue to the maintainer at **ian-cowley@users.noreply.github.com**.
2. Include a description of the vulnerability, step-by-step instructions to reproduce it, and any proof-of-concept code.
3. We will acknowledge your report within 48 hours and work with you to resolve the issue promptly.
